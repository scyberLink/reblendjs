# reblend-hooks [![npm][npm-badge]][npm]

A set of utility and general-purpose React hooks.

## Install

```sh
npm install reblend-hooks
```

## Usage

```js
import useInterval from 'reblend-hooks/useInterval'

useInterval(() => loop(), 300, false)
```
